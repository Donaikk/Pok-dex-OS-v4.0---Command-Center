# Pok-dex-OS-v4.0---Command-Center
Pokédex OS v4.0 - Command Center O Pokédex OS é uma aplicação web de alta performance que simula um centro de comando tático para treinadores Pokémon. Abandonando o visual clássico, este projeto foca em uma estética Dark Mode inspirada em interfaces militares e espaciais, utilizando a paleta de cores icônica da Pokébola: Vermelho,Branco Preto.
